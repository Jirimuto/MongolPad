English
--------------------------------------------------------------
1. copy keylayout file under ~Library/Keyboard Layouts directory
2. logout and login again.
3. from language and text select Traditional Mongolain

日本語
--------------------------------------------------------------
1. keylayoutファイルを~Library/Keyboard Layoutsフォルダにコピーする。
2. ログアウトして、再ログインする。
3. 言語とテキストからTraditional Mongolainを選択する。

简体中文
--------------------------------------------------------------
1. 把keylayout文件拷贝到~Library/Keyboard Layouts目录。
2. 重新登陆。
3. 从语言和文本选择Traditional Mongolain。
