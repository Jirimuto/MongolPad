        这是一个用Java语言编写的竖写蒙古文记事本。支持国际编码Unicode蒙古文的竖写文本编辑器, 
行进支持蒙古文的从左到右的规则。目前此编辑器还是一个评价版。我们在努力改善他的各种功能。
   目前此编辑器具有以下功能。

文件 - 
     创建文件 - 创建一个新文件
     打开 文件- 打开现有文件
     保存文件 - 保存文件
     另存文件为 - 另存文件为
     打印 - 打印全部编辑文本
     关闭 - 关闭窗口
     退出 - 退出编辑
编辑 -
     取消操作 - 取消前一个操作
     重复操作 - 重复前一个操作
     复制 - 复制选择文本到剪贴板
     剪贴 - 剪贴选择文本到剪贴板
     粘贴 - 粘贴剪贴板文本到选择区
     删除 - 删除选择文本
     查找 - 查找文本
     查找下一个 - 查找下一个
     更换 - 更换文本
     选择全部 - 选择全部文本
     日期/时间 - 插入当前日期和时间
格式  
     字体 - 改变编辑器字体
     键盘 - 键盘切换
         蒙古文键盘
         蒙古文键盘布局
     语言文字 - 切换界面文字
         蒙古文
         中文
         英文
         日文
     转换 - 转换文本
         转换成大写
         转换成小写
     自动回行切换 - 自动回行切换开关
帮助
     版本信息 - 有关蒙文记事本的版本信息

     本次版本中， 我们增加了打印功能以及改善了自动回行功能。并修改了
光标游走时的乱跑问题。
