  ᠪᠤᠰᠤᠭ᠎ᠠ ᠮᠤᠩᠭᠤᠯ ᠪᠢᠴᠢᠭᠯᠡᠭᠦᠷ 
     
ᠬᠡᠪᠯᠡᠯ : 1.1.01
ᠪᠠᠢᠢᠭᠤᠯᠤᠭᠰᠠᠨ : 2017. 08.28
ᠬᠡᠷᠡᠭᠯᠡᠵᠦ ᠪᠤᠯᠳᠠᠭ ᠮᠠᠰᠢᠨ : Windows, Mac, Linux
ᠵᠤᠬᠢᠶᠠᠭᠴᠢ : ᠵᠢᠷᠦᠮᠲᠦ@ᠳᠠᠶᠠᠭᠠᠷ ᠪᠣᠯᠤᠨ ᠠᠯᠮᠠᠰ ᠺᠣᠮᠫᠠᠨᠢ
ᠲᠤᠰ ᠪᠢᠴᠢᠭᠯᠡᠭᠦᠷ ᠦᠨ ᠲᠤᠬᠠᠢ ᠳᠠᠶᠠᠭᠠᠷ ᠠᠯᠮᠠᠰ ᠺᠤᠮᠫᠠᠨᠢ ᠳ᠋ᠤ ᠠᠯᠢᠪᠠ ᠡᠷᠬᠡᠲᠡᠢ᠃

  ᠡᠨᠡ ᠪᠤᠰᠤᠭ᠎ᠠ ᠮᠤᠩᠭᠤᠯ ᠪᠢᠴᠢᠭᠯᠡᠭᠦᠷ ᠪᠤᠯ Java ᠬᠡᠯᠡ ᠪᠡᠷ ᠪᠢᠴᠢᠭᠰᠡᠨ ᠰᠤᠹᠤᠲ ᠪᠤᠯᠤᠨ᠎ᠠ᠃
ᠡᠨᠡ ᠪᠢᠴᠢᠭᠯᠡᠭᠦᠷ ᠲᠦ ᠤᠯᠠᠨ ᠤᠯᠤᠰ ᠤᠨ ᠪᠠᠷᠢᠮᠵᠢᠶ᠎ᠠ ᠢᠶᠡᠷ ᠮᠤᠩᠭᠤᠯ ᠪᠢᠴᠢᠭ᠌ ᠢᠶᠡᠨ ᠲᠡᠮᠲᠡᠭᠯᠡᠳᠡᠭ᠌ ᠶᠤᠮ᠃ 
ᠮᠤᠩᠭᠤᠯ ᠪᠢᠴᠢᠭ᠌ ᠤᠨ ᠪᠤᠰᠤᠭ᠎ᠠ ᠪᠢᠴᠢᠯᠭᠡ ᠵᠢ ᠳᠡᠮᠵᠢᠬᠦ ᠪᠡᠷ ᠪᠠᠷᠠᠬᠤ ᠦᠭᠡᠢ ᠂ ᠡᠩ ᠢᠢᠨ ᠪᠢᠴᠢᠭᠯᠡᠭᠦᠷ ᠤᠨ ᠦᠢᠯᠡᠳᠦᠯ ᠢ
ᠪᠦᠷ ᠳᠡᠮᠵᠢᠨ᠎ᠡ ᠃ ᠡᠨᠡ ᠬᠡᠪᠯᠡᠯ ᠪᠤᠯ ᠲᠤᠷᠰᠢᠯ ᠬᠡᠪᠯᠡᠯ ᠪᠤᠯᠬᠠᠷ ᠤᠯᠠᠨ ᠳᠤᠲᠠᠭᠲᠠᠯ ᠲᠠᠢ ᠪᠠᠢᠢᠬ᠎ᠠ᠃
ᠰᠠᠨᠠᠯ ᠪᠠᠨ ᠮᠡᠳᠡᠭᠦᠯᠡᠭᠡᠷᠡᠢ ᠃
  ᠡᠨᠡ ᠪᠢᠴᠢᠭᠯᠡᠭᠦᠷ ᠲᠤ ᠳᠤᠤᠷᠠᠬᠢ ᠦᠢᠯᠡᠳᠦᠯ ᠤᠳ ᠪᠠᠢᠨ᠎ᠠ᠃
